# Terms of Service

Effective date: 2026-06-07

Net Boost Agent provides local network diagnostics, benchmarking, recommendation, and report generation tools for agent environments.

## Intended Use

The agent helps users understand network conditions, identify likely issues, compare before and after measurements, and decide whether low-risk fixes are appropriate.

## No Guaranteed Improvement

Network performance depends on hardware, Wi-Fi conditions, ISP routing, remote servers, local software, and many other factors. The agent does not guarantee faster internet. It reports improvement only when before and after evidence supports that conclusion.

## User Control

Diagnostics and benchmarks are read-only by default. Any system-changing action requires preview and explicit user confirmation. Users are responsible for deciding whether to apply recommended changes.

## Third-Party Tools

The agent may use external tools such as Ookla Speedtest CLI when available. Those tools are provided by their respective owners and may have their own license terms, privacy rules, or network behavior.

## Local Artifacts

The agent may save local benchmark, report, and audit files so users can compare results and review actions. Users are responsible for managing or deleting those local files.

## Limitations

The agent is diagnostic and advisory software. It is not a replacement for ISP support, enterprise network administration, router firmware updates, or professional troubleshooting for critical systems.

## Contact

For this local development package, contact the Net Boost Agent maintainers through the repository or distribution channel where the package was provided.
